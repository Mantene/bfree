The bFree Application

The bFree application extracts the data and organization from a Blackboard™ course "Export" (or "Archive") zip file, and creates either a web site or a directory hierarchy with the extracted information. The web site or the folders exhibit the same structure and names as the original Blackboard™ course.

In addition, the web site uses page templates that can be customized to have an appearance and "branding" suitable for any organization or person.

bFree thus allows course authors to:
  • conveniently retrieve texts and other materials that might not otherwise be available outside of Blackboard™
  • produce independent course web sites with the same content and structure
  • easily share the course data or web site with others

How to Export from Blackboard™

Given that you have administrative privileges for a course (and you do if you are the course's author), navigate into the Blackboard™ course, then:
  • click on the "Control Panel" link (in the left-hand navigation area) to open the Control Panel
  • in the "Course Options" area of the Control Panel, click on "Export Course" (or "Archive Course")
  • on the "Export Course" (or "Archive Course") page, select the items you would like to export (so that bFree can extract them)
  • check the "Settings" box, which provides some basic course information
  • then click the "Submit" button
Another page appears, displaying a link to the Export (or Archive) zip file; save this to your computer or other file system. Then click the "OK" button on that page.

Using bFree

When you open bFree, you will see a title and message at the top, a large display area in the middle, and several buttons at the bottom. The (nearly empty) display area will hold an outline of the Export (or Archive) zip that you choose to extract.

The Buttons

The labels on the buttons, and the actions prodcued by the buttons are:
  • Open an exported zip file ... asks you to choose a zip file from your computer or other file system; the contents will then be displayed
  • Show actual content only some course sections may have no data (or bFree does not handle them yet); this button takes them off the display
  • Extract as files/folders ... asks you to designate a folder, then extracts the course content into that folder, creating a folder hierarchy that parallels the organization of the original course
  • Create a basic web site ... asks you to designate a folder, then extracts the course content into that folder as a set of web pages and files that parallel the organization of the original course; the pages are made using a set of default templates
  • Obtain example templates ... asks you to designate a location at which a folder containing a set of the default templates will be deposited; you can modify the templates, then use them to generate a custom web site
  • Create a custom web site ... asks you to locate a set of page templates, then designate a folder in which a customized web site will be created from the course information

The Display

After you open an Export (or Archive) zip file, its contents and organization will be displayed as a standard hierarchy of folders, subfolders and files in the large display area in the middle of the window.

For each item in the hierarchy, there are entries in the table on the right-hand side. The table columns are:
  • Type tells what the entry represents within the actual Blackboard™ course:
    • "Course" is the overall course itself
    • "Section" is a course section as listed in the main (left-hand) navigation menu within the course
    • "Item" is one of the several content items within a section, usually consisting of a title (also the item's title in this display), some text, and perhaps an attachment
    • "Text" is the descriptive text of an item
    • "File" is any file attached to an item
    • "Link" is a URL associated with an item
    • "Info" is a staff information entry
    • "Folder" is a folder that can contain items
    With further development, there will be other types supported as well.
  • Extract? when this box is checked, this item is designated to be extracted
  • Created and Upddated are dates, as indicated (this information cannot always be established)
Since bFree maintains the folder hierarchy when extracting content, it insists on checking the "Extract?" boxes in the path from "Course" to anything you choose to extract; this makes checking these boxes a little odd:
  • unchecking or checking a box at a higher level automatically unchecks or checks all its subordinates
  • unchecking a lower-level box does not affect higher-level entries
  • checking a lower-level box automatically checks the boxes in the path up to the "Course" entry.

Customizing the Templates

Any web site that bFree creates is generated using a small set of simple HTML templates. Each template has the basic HTML for a particular page layout, plus a keyword or two indicating where content that is produced by bFree should be inserted. The keywords are of the form "BflatUnzip$xxx$", and each different keyword refers to a different kind of content.

You may obtain a folder containing these page templates by clicking the appropriate button in the bFree window. Then you can modify or expand the pages to suit your needs or tastes, being sure to keep the keywords where you would like the content to be inserted.

The restrictions on your design, if you wish to generate the web pages using your modified templates are:
  • the templates must keep the names as given
  • all the templates must be present
  • the "framesets" defined in the index page must remain
  • a page accepts only the data indicated by the keyword(s) in the example page
Of course, you can extract the content once, then use it in any way you please, rather than relying on the templates.
